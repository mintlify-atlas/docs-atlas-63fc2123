## Overview

WhatsApp Audio Transcriber is a powerful bot that listens to your WhatsApp conversations and automatically transcribes voice messages when you react to them with a specific emoji. Built with Node.js and powered by OpenAI's Whisper speech recognition model, it provides accurate, multi-language transcription directly in your WhatsApp chats.

## How It Works

<Steps>
  <Step title="Start the Bot">
    Run the application and scan the QR code with your WhatsApp mobile app to authenticate.
  </Step>
  <Step title="React to Voice Messages">
    When you receive a voice message, react to it with your configured emoji (default: 🤖).
  </Step>
  <Step title="Get Transcription">
    The bot automatically downloads the voice message, processes it through Whisper, and replies with the transcription.
  </Step>
</Steps>

## Technology Stack

This project is built with:

- **[whatsapp-web.js](https://github.com/pedroslopez/whatsapp-web.js)** - WhatsApp API client using Puppeteer
- **[smart-whisper](https://github.com/JacobLinCool/smart-whisper)** - Node.js wrapper for whisper.cpp
- **[Express](https://expressjs.com/)** - Web server for QR code display
- **[FFmpeg](https://www.ffmpeg.org/)** - Audio processing and conversion

## Quick Example

Once configured, using the bot is as simple as:

1. Receive a voice message in WhatsApp
2. React to it with 🤖 (or your custom emoji)
3. Wait a few seconds for processing
4. Receive the transcription as a reply

The bot handles all the audio conversion, processing, and transcription automatically.
